To import a specific version of the Common DLLs, select:

"Assets => Import Package => Custom Package..."

then navigate to the package you would like to import.
The DLLs should automatically be updated and their references should be kept.


Compilation flags:

UNITY_2018_3_OR_OLDER	= Compiled to function with 2018.3 or older versions

UNITY_2018_2_OR_OLDER	= Compiled to function with 2018.2 or older versions

UNITY_2018_1_OR_OLDER	= Compiled to function with 2018.1 or older versions

NET_LEGACY				= Compiled with .NET 3.5 Unity full Base Class Libraries (legacy .NET support)
